# AppCalificaciones
Aplicación de consola para calcular calificaciones.

Pasos Para ejecutar la aplicaión de consola:
1. Abrir el proyecto en Visual Studio.
2. Presionar el botón verde para comenzar la ejecuión.

Nota: Para visualizar la información de base de datos de forma rápida y sencilla una vez que se hayan guardado registros
se puede cargar en la página https://inloop.github.io/sqlite-viewer/
el archivo SQLiteDBLocal.db que se guardará dentro del directorio ../AppCalificaciones\bin\Debug\net6.0 del proyecto
